# Noctua UCA

Noctua UCA es una aplicación móvil diseñada exclusivamente para los estudiantes de la Universidad José Simeón Cañas. Esta aplicación sirve como un portal integral que proporciona herramientas útiles para mejorar la experiencia universitaria de sus estudiantes. Desarrollada con Kotlin para Android, la aplicación se conecta a un backend en Express.js y utiliza MongoDB como sistema de gestión de bases de datos. 

El objetivo de Noctua UCA es ofrecer a los estudiantes una plataforma fácil de usar para acceder a información esencial y realizar diversas actividades académicas. Entre los módulos disponibles se incluyen la búsqueda de perfiles, la edición de perfil personal, horarios de laboratorios, seguimiento de pensum, generación de código de carnet QR, y mapas guías del campus. Estos módulos están diseñados para facilitar la navegación diaria y la planificación académica en el campus.

## Características

- **Búsqueda de Perfiles**: Permite a los estudiantes buscar y visualizar perfiles de otros estudiantes para fomentar la colaboración y la comunicación.
- **Edición de Perfil Personal**: Los usuarios pueden actualizar y mantener su información personal actualizada fácilmente.
- **Horarios de Laboratorios**: Acceso rápido a los horarios de los laboratorios disponibles, permitiendo a los estudiantes planificar sus actividades prácticas.
- **Seguimiento de Pensum**: Ayuda a los estudiantes a seguir su progreso académico y planificar sus semestres futuros.
- **Código Carnet QR**: Genera un código QR personal que los estudiantes pueden utilizar como carnet digital para diversos servicios dentro del campus.
- **Mapas Guías del Campus**: Ofrece mapas interactivos para ayudar a los nuevos y actuales estudiantes a navegar por el campus universitario.

## Tecnologías Utilizadas

- **Frontend**: Kotlin
- **Backend**: Express.js
- **Base de Datos**: MongoDB
- **Otras tecnologías**: Retrofit para la conexión de red, Jetpack Compose para la interfaz de usuario.

## Instalación

Para comenzar a usar Noctua UCA en tu dispositivo Android, sigue los siguientes pasos:

1. Clona este repositorio en tu máquina local usando: https://github.com/PDM-24/proyecto-noctua-uca.git
2. Abre el proyecto en Android Studio.
3. Ejecuta el servidor backend siguiendo las instrucciones en la carpeta del servidor.
4. Compila y ejecuta la aplicación en tu dispositivo o emulador.
